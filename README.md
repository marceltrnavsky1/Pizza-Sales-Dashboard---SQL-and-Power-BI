## 📊 Portfolio Project – Power BI & SQL

# This project is primarily focused on developing skills in Power BI with some SQL practice included.

## 🔍 Project Overview
Import Data in SQL
SQL Queries for KPI's
SQL Queries for Daily and Monthly Trend
SQL Queries for % of Sales by Category & Size
SQL Queries for Top 5 & Bottom 5
Connecting Power BI to SQL DB
Data Cleaning using Power Query
KPI's Building
Data Processing in Power BI
Building Daily and Monthly Trend Charts
Building % of Sales by Category & Size
Adding Insights
Worst and Best Sellers Dashboard
Add Navigator Buttons

## 🎯 Objective
The objective of this project was to gain hands-on experience with report creation in Power BI and to understand the full data workflow – from raw data to final presentation.

## 📈 Outcome
The final result is a complete interactive dashboard that allows users to:

filter and analyze data based on different criteria
track key metrics
gain quick insights from the data
